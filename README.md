## 실행

```bash
npm install
npm run dev
```

## 빌드 & 미리보기

```bash
npm run build
npm run preview
```

## 스크립트

* `dev` — 로컬 개발
* `build` — 프로덕션 빌드
* `preview` — 빌드 확인
* `lint` — ESLint 검사
